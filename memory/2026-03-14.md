## 修复 Web 面板离线问题
- 问题：Gateway 健康检查显示离线
- 原因：日志显示之前有 `gateway closed (1006 abnormal closure)` 异常断开
- 解决：执行 `openclaw gateway restart` 重启 Gateway
- 结果：状态恢复正常，面板显示在线

## 学到的
- Windows 系统上用 `timeout /t` 来延时
- Gateway 绑定的 PID 是 13868（node.exe）
- 健康检查断开后可能需要手动重启才能完全恢复状态显示
