# 农业养殖传感器需求清单
# 创建时间：2026-03-26 09:30
# 需求来源：QQ用户

## 1. 水产养殖传感器
### 监测类型：
- pH值传感器
- 温度传感器
- 盐度传感器
- 溶解氧传感器

## 2. 牛羊养殖传感器
### 环境监测：
- 环境温湿度传感器
- 氨气（NH₃）传感器
- 二氧化碳（CO₂）传感器
- 甲烷（CH₄）传感器
- 光照强度传感器

### 可穿戴监测：
- 活动量/反刍监测传感器
- 体温传感器

## 3. 禽类养殖传感器
### 环境监测：
- 空气温度传感器
- 相对湿度传感器
- 氨气（NH₃）传感器
- 二氧化碳（CO₂）传感器
- 静压与风速传感器（负压通风评估）
- 光照强度与光周期传感器
- 脚垫湿度传感器

## 4. 猪养殖传感器
### 环境监测：
- 空气温度传感器
- 相对湿度传感器
- 氨气（NH₃）传感器
- 硫化氢（H₂S）传感器
- 二氧化碳（CO₂）传感器
- PM2.5/粉尘浓度传感器
- 光照强度传感器

## 采购要求：
1. 需要厂家信息
2. 产品型号
3. 通讯协议（尽可能选择大品牌）
4. 优先选择：霍尼韦尔、西门子、博世、欧姆龙、施耐德等国际品牌
